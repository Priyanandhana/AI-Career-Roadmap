# AI-Powered Career Roadmap Generator for Students

## Project Overview
This project is an intelligent web-based application designed to generate personalized career roadmaps for students based on their background, experience level, and career goals. Using Large Language Models (LLMs) and Streamlit, the system analyzes user inputs and produces a structured, step-by-step plan to help students achieve their desired roles.

The system acts as an AI Career Mentor that provides:

- Skill-building guidance  
- Learning stages  
- Tools and technologies to master  
- Suggested projects  
- Timeline-based roadmap  
- Motivation and career tips  

## Objectives
- Help students understand the path to their target career  
- Provide a clear structured roadmap  
- Reduce confusion in career planning  
- Encourage goal-oriented learning  
- Automate career guidance using AI  

## Target Users
- Beginners entering a new career field  
- Students unsure about career progression  
- Job seekers wanting structured guidance  
- Professionals planning a role switch  

## System Workflow
1. Student enters details:  
   - Name  
   - Current Experience Level  
   - Target Job Role  
   - Career Goal  
2. Input is sent to the AI model.  
3. AI analyzes and generates a roadmap.  
4. Streamlit displays the roadmap in a structured format.  

## Core Features
- Personalised Career Roadmap  
- Skill Development Plan  
- Tool Recommendations  
- Project Ideas  
- Timeline Planning  
- Motivation Section
